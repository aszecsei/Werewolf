# Werewolf

This is a script to test AIs for the party game Werewolf, the rules for which are available [here](http://www.playwerewolf.co/rules/). By default, the games are created with 12 AIs, the roles for which are randomly assigned.

A sample log of a game follows (with the Debug option enabled to view the game):

```
Player #9 (Villager) was killed in the night.
Player #0 says that Player #10 is a Villager
Player #8 believed Player #0's statement that Player #10 is a Villager!
Player #2 believed Player #0's statement that Player #10 is a Villager!
Player #11 (Villager) was lynched by an angry mob!
Player #1 (Villager) was killed in the night.
Player #0 says that Player #7 is a Villager
Player #3 believed Player #0's statement that Player #7 is a Villager!
Player #6 believed Player #0's statement that Player #7 is a Villager!
Player #5 believed Player #0's statement that Player #7 is a Villager!
Player #8 (Villager) was lynched by an angry mob!
Player #7 (Villager) was killed in the night.
Player #0 says that Player #6 is a Werewolf
Player #10 believed Player #0's statement that Player #6 is a Werewolf!
Player #5 believed Player #0's statement that Player #6 is a Werewolf!
Player #6 says that Player #6 is a Villager
Player #6 says that Player #0 is a Werewolf
Player #0 says that Player #0 is a Villager
Player #4 believed Player #0's statement that Player #0 is a Villager!
Player #3 believed Player #6's statement that Player #0 is a Werewolf!
Player #3 believed Player #0's statement that Player #6 is a Werewolf!
Player #4 believed Player #0's statement that Player #6 is a Werewolf!
Player #6 (Werewolf) was lynched by an angry mob!
The doctor managed to save Player #10!
Player #4 says that Player #10 is a Villager
Player #0 says that Player #2 is a Villager
Player #4 believed Player #0's statement that Player #2 is a Villager!
Player #11 (Villager) was lynched by an angry mob!
Player #4 (Doctor) was killed in the night.
Player #0 says that Player #3 is a Werewolf
Player #3 says that Player #3 is a Villager
Player #3 says that Player #0 is a Werewolf
Player #0 says that Player #0 is a Villager

The villagers have won!
```
